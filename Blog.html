4. Does it matter to use relative size(%,em,rem) one styling the size of an element so that the element is responsive ? Why ? [write your argument]

Yes it does, because:

* % is not a length that is relative to viewport’s width or height. Instead, it is a percentage of the parent element’s width or height. Percentage units are often useful to set the width of margins.

* EM is relative to the parent element’s font size, so if it is to scale the element’s size based on its parent’s size, use EM.

* REM is relative to the root (HTML) font size, so if it is to scale the element’s size based on the root size, no matter what the parent size is, use REM. If EM used and find some sizing issues due to lots of nested elements, REM will probably be the better choice.

Mathimatical relation:(using the default px)

100% = 16px

1em = 16px

1rem = 16px

6. Explain in detail(in technical terms) the specific changes in css3

1. RGBA: CSS 3 introduces a modification and added opacity manipulation to this property. RGB has been transformed to RGBA (Red Green Blue Alpha channels), which simplifies how to control the opacity of elements.
{background: rgba(217, 127, 185, .5); }

2. Round corners: CSS 3 addresses the border-radius property, which gives the same rounded-corner effect without writeing all the code.
{ border: 2px solid orange; border-radius : 25px; width: 100px; padding: 10px; text-align:center; }

3. Border image: is the ability to swap out a border with an image. The property border-image allows to specify an image to display instead of a plain solid-colored border.

4. Transform: CSS 3 introduced a property called transform, which enables rotating Web elements on a webpage.
{ -moz-transform: rotate(180deg); -webkit-transform: rotate(180deg);}

5. Web Fonts: CSS 3 also update embedding any custom font on a webpage. Fonts are dependent on the client system and Web pages can render only fonts that are supported by the browser or the client machine. By using the @font-face property, it is able to include the font from a remote location and can then use it.

6. Selectors: CSS 3 introduces some new selectors. Using these selectors its able to choose DOM elements based on their attributes. So no need to specify classes and IDs for every element. Instead, utilize the attribute field to style them.

7. Box Shadow: A box shadow allows to create a drop shadow for an element. Usually this effect is achieved using a repeated image around the element. However, with the property box-shadow this can be achieved by writing a single line of CSS code.
{box-shadow:3px 3px 3px 2px #797979}

8. Text Shadow: The new text-shadow property allows you to add drop shadows to the text on a webpage.
{ text-shadow: #aaa 2px 2px 2px; }

there are also another changes in CSS3 like, animation and transition, calculating values with calcC, generate content and countets, box sizing, media queries , multiple backgrounds , 3D transforms, Multicolumn Layout.
